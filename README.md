# A-Comprehensive-AI-MODELS-TRAINING-Framework-for-Flood-Detection-
End-to-end satellite flood detection toolkit — acquire, annotate, and train multi-modal transformer models, then map flood-affected buildings worldwide.

This Project was done in collaboration with Narss - National Authority For Remote Sensing & Space Sciences in Egypt
